# GreenQuest Mobile-First Prototype

A clickable React prototype for a gamified environmental action platform.

## Included screens

- Youth home dashboard
- Environmental challenge discovery
- Challenge details
- Three-step mission flow
- Quiz interactions and evidence upload
- Individual and school leaderboards
- Green Passport / impact portfolio

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite. On desktop, the app appears inside a mobile device frame. On phones, it uses the full screen.

## Build for production

```bash
npm run build
npm run preview
```
